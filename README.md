# Hello Kitty Buddy 🎀

A virtual pet app featuring Hello Kitty with interactive feeding, petting, napping, and mini-games.

## Deploy to Vercel

```bash
npm i -g vercel
vercel login
vercel --prod
```

Or push to GitHub and connect to Vercel.

## Development

```bash
npm install
npm run dev
```

## Features

- 🍓 Feed - Fill hunger
- 💕 Pet - Increase happiness
- 💤 Nap - Restore energy
- 🎮 Play - Mini-game to catch goodies
- XP & Level system with localStorage persistence
